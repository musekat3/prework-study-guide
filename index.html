HTML

  • The head element contains information, or Metadata, that helps the browser to render the page correctly.
  • The body element represents the visible content shown to the user.
  • title - defines the title of a webpage
  • style - defines CSS styles associated with the website
  • base - defines the base URL of a webpage
  • link - connects an external source to the HTML document
  • meta - deines metadata such as character set, description, keywords, author, and viewpoint
  • script - embeds client-side scripts in the HTML document
  • /head - closing tag
  • !DOCTYPE html - tells the browswer what document it should expect when opened
  • div - to create a division in the page
  • span - to create an inline division
  • p - paragraph block or text
  • ul - unordered or bullet form
  • li - list items
  • img - contains information about images that are displayed on website
  • src - defines the location of the image
  • br - creates an empty space
  • a - (anchor) creates links to the same webpage or other webpages
  • href - attributes points to the URL for the link
  • footer - contains author, copywrite, contact, sitemap, and navigation

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element
  • There are three ways to use CSS to style a webpage: inline, internal, and external
  • In the CSS rule, property has to have value and vice versa to make a declaration
  • You can use the website htmlcolorcodes.com for color references

Git

  • ls - opens computer directory
  • cd bootcamp - opens bootcamp directory
  • cd prework-study-guide - opens prework-study-guide directory
  • git add -A - stage changes for Git Flow
  • git commit -m "your message here" - describes the changes that you have made that you want to note
  • git pull origin main - updates the local branch for any changes
  • git push origin "branch name" - pushes the changes to the remote respository
  • git status: checks what branch we are cuurently on
  • git checkout -b branch-name: creates a new branch and switches to it

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • array - single variable that is used to hold a group of data, typically that are related in some way
  • The first item of an array has an index of 0
  • function - a simple set of instructions that tells the computer how to perform a certain task
